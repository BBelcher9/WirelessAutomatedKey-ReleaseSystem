# Wireless Automated Key-Release System

## Demonstration Video:
https://www.youtube.com/watch?v=V6xTp-4p8c4

## Overview
This project is a custom automation systems that combines 3D modeling, embedded programming, and electronics to create a LAN remote controlled key dropping device. This demonstrated an end-to-end engineering workflow from mechanical to programming.

Using an http request, you can control preset movements of the device to release keys remotely.

See Assembly Guide PDF file for construction instructions.
